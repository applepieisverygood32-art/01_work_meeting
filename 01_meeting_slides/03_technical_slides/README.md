# 技術分析簡報 Prompt

本資料夾用於整理：

污染調查、監測分析、技術方法、環境流布、污染傳輸及專業技術內容相關之簡報 Prompt。

---

# 主要用途

- 技術分析簡報
- 監測成果
- 環境流布分析
- 污染傳輸途徑
- 底泥調查
- 土壤調查
- 水質監測
- 技術方法說明

---

# 常見簡報類型

- PFAS 農地污染途徑
- PBDE 環境流布
- 土壤與底泥分析
- 渠道污染調查
- 縮時膠囊技術
- 監測方法比較

---

# 使用原則

1. 保持技術正確性。
2. 不得過度推論污染來源。
3. 圖表應保持可閱讀性。
4. 技術內容應兼顧簡報可理解性。
