# 工作會議 AI 協作模板庫

本 repository 用於整理工作會議、專案討論會、業主簡報及內部工作會議相關之 AI 協作 Prompt 與簡報模板。

本資料庫以「快速生成可正式使用之會議資料」為核心目的，協助承辦人提升會議資料整理、簡報製作及論述準備效率。

---

# 主要用途

- 工作會議簡報生成
- 業主會議簡報
- 討論會資料整理
- 會議紀錄整理
- 會議簡報圖卡
- 演講稿生成
- 決策論述整理
- 工作進度簡報

# 使用原則

1. AI 內容須由承辦人確認。
2. 不得直接引用未確認之制度內容。
3. 圖卡與流程圖需確認邏輯正確。
4. 重要政策論述應保留正式用語。
5. 簡報應以「決策閱讀」為核心。
